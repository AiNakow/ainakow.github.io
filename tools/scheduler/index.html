参数设置

💡 如何获得更优的赛程（交手次数差更小）
调高级参数（下方折叠区）：
• 增大 maxSteps（如 60000 → 120000）——给模拟退火更多迭代时间，解更趋近最优；
• 增大 restarts（如 8 → 16）——多轮随机重启，从更多起点中挑最好的解；
• 微调 cooling（如 0.99975 → 0.9999，更慢降温）——温度降得慢，搜索更细致，但需配合更大的 maxSteps。
调基础参数：
• 增大每队场数 m或减小队伍数 n——比赛场次更多，每对队伍交手机会增加，更易把差距压到 ≤ 上限，也更容易让所有队伍都至少交手一次。
提示：每次生成带随机种子，换种子或点「再生成一个」也会得到不同质量的解；固定同一种子可复现某个解。
